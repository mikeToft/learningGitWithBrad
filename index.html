Learning Git and Github

Intro

I have been learning about Git and Github from the Master, Brad Traversy. I watched his YouTube video entitled Git & GitHub Crash Course For Beginners. The notes here are from that video.

Notes:

  • Git is a version control system
  • Can revert back to specific versions of code
  • Local an remote repositories
  • A commit takes a snapshot of the code at that time
  • Can visit any commit at any time
  • You can stage files before committing

Git commands

Command Purpose
$ git init Initialise a local git repo
$ git add Add file to the index
$ git add *.html Adds all the html files
$ git add . Adds everything
$ git status Check status of working tree
$ git commit Commit changes in index
$ git commit -m ’dev notes’ Commit with dev comment
$ git push Push to remote repo
$ git pull Pull latest from remote repo
$ git clone 'github clone URL' Clone a repo into a new directory
$ git -- version Dash dash, get version of git on the computer
$ git rm cached 'file name' Removes the file
$ git branch 'branch name' Creates a branch
$ git checkout 'branch name' Opens the branch
$ git checkout master Return to master branch
$ git merge 'branch name' Adds the named branch and its changes to the master
